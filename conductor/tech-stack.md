# Tech Stack: CarbonIQ

## Core Components
- **Programming Language:** Python 3.9+
- **Frontend Framework:** Streamlit (v1.30+) for a responsive dashboard.
- **Data Visualization:** Plotly (Express and Graph Objects) for interactive charts.

## Data & Analytics
- **Data Manipulation:** Pandas and NumPy for large-scale logistics data processing.
- **Carbon Modeling:** GLEC Framework implementation for high-accuracy emissions estimation.
- **Carrier Benchmarking:** EPA SmartWay dataset for efficiency leaderboards.

## Artificial Intelligence
- **LLM Integration:** Hugging Face Inference API for natural language queries.
- **Model:** Qwen 2.5 72B Instruct for sustainability-specific intelligence.
- **Integration:** Context-aware assistant that utilizes fleet metrics for real-time insights.

## Development & Infrastructure
- **Dependency Management:** pip via `requirements.txt`.
- **Environment Configuration:** Python-dotenv for managing sensitive API tokens.
- **Styling:** Custom CSS and Streamlit components for a dark glassmorphic UI.
