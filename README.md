# CarbonIQ

CarbonIQ is an enterprise-grade Carbon Intelligence Dashboard for road freight. It provides real-time emissions monitoring, shipment estimation, lane analytics, and what-if scenario modeling, all supported by a persistent AI Assistant.

## Quick Start

1. Install dependencies: `pip install -r requirements.txt`
2. Set `HF_API_TOKEN` in a `.env` file.
3. Run the app: `streamlit run app/app.py`
